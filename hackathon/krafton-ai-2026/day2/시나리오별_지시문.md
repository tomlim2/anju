## 시나리오 D: offset 찾았으나 확신 부족

```
offset 확신도 높이기 작업.

작업 디렉토리: /Users/younsoolim/Desktop/www/anju/hackathon/krafton-ai-2026/day2/
후보 offsets: [리스트]

집중 검증:
1. Training data 역검증 — 비트별 정확도 (전체 평균 + position별)
2. Bootstrap: 2000개에서 1000개 랜덤 샘플 x 100회 → offset 재발견율
3. Leave-one-out: offset 하나씩 빼보면서 정확도 변화 (모든 offset이 필요한지)
4. 추가 offset: 현재 set에 d=1..64 하나씩 추가해보며 정확도 개선 여부
5. confidence_analysis.json에 저장

80% 이상이면 확정, 아니면 offset set 수정 제안.
```
