# ImageMagick 스프라이트 시트 생성기 사용 설명서

이 .bat은 여러 개의 이미지를 하나의 스프라이트 시트로 자동 변환해주는 도구입니다.

## 사용 방법

### 준비 사항
- ImageMagick 프로그램이 컴퓨터에 설치되어 있어야 합니다.
- 설치되어 있지 않다면 https://imagemagick.org/archive/binaries/ImageMagick-7.1.1-47-Q16-HDRI-x64-dll.exe을 다운로드 해주세요.

### 주요 폴더
#### input
- 배치 파일이 있는 같은 폴더에 'input' 폴더를 만들어주세요.
- 'input' 폴더 안에 변환하고 싶은 이미지들이 담긴 하위 폴더들을 만들어주세요.
	예: input\캐릭터1\, input\캐릭터2\ 등

#### output
- 변환된 스프라이트 시트는 'output' 폴더에 저장됩니다.
- 파일 이름은 원본 폴더 이름에 최대 프레임 갯수가 추가된 형태입니다.
	예: output\캐릭터1_144.png, output\캐릭터7_111.png 등

### 실행 방법
- generate_sprite_sheet.bat 파일을 더블 클릭하면 자동으로 실행됩니다.
- 프로그램이 실행되면 'output' 폴더가 자동으로 생성되고, 그 안에 스프라이트 시트 이미지들이 만들어집니다.

## 특징
- 최대 144개의 이미지를 하나의 스프라이트 시트로 변환합니다.
- 각 이미지는 80x80 픽셀 크기로 변환됩니다.
- 최종 스프라이트 시트의 크기는 1024x1024 픽셀입니다.
- PNG와 JPG 형식의 이미지를 모두 지원합니다.

## 참고 사항
- 이미지 파일의 순서는 파일명 알파벳 순으로 정렬됩니다.
- 너무 많은 이미지가 있을 경우 처음 144개만 처리됩니다.
- 문제가 발생하면 ImageMagick이 올바르게 설치되었는지 확인해보세요.
